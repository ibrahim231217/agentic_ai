## 💡 KEY INSIGHTS

### Architecture Decision
```
Why this design?

Frontend (Streamlit)
  ↓ Reason: Fast to build, professional UI
  
Agents (Modular design)
  ↓ Reason: Each agent is independent, easy to extend
  
APIs (Free tier)
  ↓ Reason: No costs, good for prototyping
  
File Handlers (Utilities)
  ↓ Reason: Reusable, testable, maintainable
```

### Technology Choices
- **Streamlit**: Rapid development, excellent UI
- **OpenRouter**: Access to many free models
- **Gemini**: Better for summarization specifically
- **PyPDF2 + python-docx**: Standard libraries for file handling
- **python-dotenv**: Secure credential management
